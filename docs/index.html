A very extensive set of extension methods that allow you to more naturally specify the expected outcome of a TDD or BDD-style unit tests. Works with MSTest, xUnit, NUnit, Gallio, MBUnit, MSpec and NSpec. Very detailed and self-explanatory failure messages. Parses the C# code to identify the subject-under-test. Allows asserting the equivalency of (collections of ) deep object graphs. Fully thread-safe. Specialized assertions for `async` code. Execution time assertions. Type, `[attribute]` and assembly-level dependency assertions. Assert on raised `event`s and usage of `INotifyPropertyChanged`. Represent dates and times as `20.September(1973).At(8, 1)`.
